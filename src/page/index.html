데이터 소스
Excel 시트 → JSON 자동 변환

각 카테고리(캐릭터 / 장비 / 아이템)를 엑셀 시트로 관리하고, 스크립트로 JSON으로 변환한 뒤 GitHub Pages에 올려서 바로 반영합니다.

엑셀 관리

시트에서 필드 추가/수정 후 JSON만 다시 내보내면 됩니다.

정적 호스팅

GitHub Pages에서 HTML + JS + JSON만으로 동작합니다.

메인 메뉴

각 메뉴는 나중에 별도 HTML / JS로 구성해서 sstoy처럼 독립된 툴 페이지로 만들 수 있습니다.

C
캐릭터 DB

캐릭터 스탯, 속성, 역할, 태그를 기준으로 검색/필터 가능한 데이터베이스. 엑셀 시트를 기반으로 JSON을 생성해서 사용합니다.

E
장비 DB

무기 / 방어구 / 특수 장비 옵션과 추천 캐릭터를 정리하는 페이지. 우선 UI 뼈대부터 만들고 나중에 실제 데이터를 연결할 수 있습니다.

I
아이템 DB

재료, 강화석, 소모품 등 각종 아이템을 한 눈에 볼 수 있는 DB. 드랍처와 사용처도 같이 표기하는 용도로 설계할 수 있습니다.

T
도구 / 계산기

빌드 시뮬레이터, 재화 계산기, 파밍 경로 추천 등 부가적인 도구를 모아두는 영역입니다.
